# Mongodb with node js

## Build Setup

``` bash
# download and install node js
https://nodejs.org/en/

# download mongo compass community
https://www.mongodb.com/products/compass

# For creating database use create.js file and to check output run  
node create.js

# For creating collection use collection.js file and to check output run  
node collection.js

# To upload any json file (or) any other file use upload.js file and to check output run  
node upload.js

# To check particular info form file use query.js file and to check output run  
node query.js

# To drop collection form database use drop.js file and to check output run  
node drop.js
```
